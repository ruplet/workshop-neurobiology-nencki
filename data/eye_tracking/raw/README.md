Files: `data/eye_tracking/raw/KFD*.txt` (tab-separated)

Columns:
- `trial`: trial identifier
- `buf_index`: row/sample index from acquisition buffer
- `pos_time`: gaze timestamp (ms)
- `pos_x`: gaze x-coordinate
- `pos_y`: gaze y-coordinate
- `pup_time`: pupil measurement timestamp (ms)
- `pup_diameter`: pupil size estimate
- `marker`: event marker flag (dataset-specific)
